# HTTP vs HTTPS Demo

A simple animation-based project that demonstrates the difference between HTTP and HTTPS communication.

## Features
- Visual data flow: User → Browser → Server
- HTTP shows unencrypted data (visible to attackers)
- HTTPS shows encrypted data (Base64 simulation)
- Animated packet movement
- Node highlighting

## Tech Used
- HTML
- CSS
- JavaScript

## How to Run
Just open `index.html` in a browser.

## Purpose
To visually teach how HTTPS protects data compared to HTTP.
